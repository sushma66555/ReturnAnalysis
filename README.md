# Return Data Analysis Dashboard (Power BI)

## 📌 Project Overview
This project contains a Power BI dashboard that analyzes **Return Data by Year and Region**.  
It helps to identify:
- Year-wise trends in return data
- Regional distribution and comparison
- Key insights for business decision-making

## 🛠 Tools Used
- **Power BI Desktop** – for building the dashboard
- **Excel/CSV** – as the source dataset
- **GitHub** – for project version control

## 📊 Dashboard Highlights
- Interactive filters for **Year** and **Region**
- KPIs showing total returns and return percentage
- Trend charts showing year-wise growth/decline
- Regional comparison through bar/column charts

## 📂 Repository Structure
- `ReturnData.pbix` → Power BI report file
- `Dataset/` → Source dataset (if shared)
- `Screenshots/` → Dashboard preview images
- `README.md` → Project documentation

## 🚀 How to Use
1. Download the `ReturnData.pbix` file.
2. Open it using **Power BI Desktop**.
3. Explore the dashboard using slicers and filters
